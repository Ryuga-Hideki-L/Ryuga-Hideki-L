Бэкенд-разработчик, в основном на **Go** — сервисы, REST/WebSocket/gRPC API, реалтайм-логика
и базы под ними. Иду к тому, чтобы делать это профессионально; ценю код, который «скучный»
в хорошем смысле: предсказуемый, покрытый тестами, которому легко доверять. Когда проекту
нужен весь стек — беру фулстек: TypeScript/React сверху, SQL снизу.

### Сейчас делаю — LifeRPG

Пиксельная RPG, которая превращает реальное саморазвитие — шахматы, фокус-сессии,
тренировки, слепую печать — в прокачку персонажа. Бэкенд на **Go**: REST + WebSocket
карточные бои, бесконечная роглайк-башня с ветвящейся картой, лут и сундуки, серверный
античит (клиенту не доверяем). Клиент на React/TypeScript, SQLite в разработке со
схемой под Postgres.
